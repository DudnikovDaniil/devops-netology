# devops-netology

Репозиторий для домашних заданий по курсу DevOps

## Описание
Этот репозиторий создан для выполнения домашних заданий в рамках курса по DevOps.

## .gitignore для Terraform

Файл `.gitignore` настроен по образцу [Terraform.gitignore](https://github.com/github/gitignore/blob/main/Terraform.gitignore)

В файле `.gitignore` используются следующие шаблоны:

- `**/.terraform/*` — игнорируется всё содержимое папок `.terraform` на любом уровне вложенности
- `*.tfstate` — игнорируются любые файлы с расширением `.tfstate`
- `*.tfstate.*` — игнорируются любые файлы с расширением, начинающимся на `.tfstate`
- `crash.log` — игнорируется файл лога
- `override.tf` — игнорируются файлы переопределения
- `.terraformrc` — игнорируется файл конфигурации
- `!example_override.tf` — этот файл НЕ игнорируется (исключение)

**Синтаксис правил .gitignore:**
- `*` — соответствует любой строке (любое количество любых символов)
- `**` — соответствует любому количеству вложенных папок
- `!` — исключение (отмена игнорирования)
- `/` — разделитель папок; в начале строки означает корневую папку репозитория
